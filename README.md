# Enterprise Information Security & Acceptable Use Policy (AUP)

## 📌 Overview
This repository contains a comprehensive, enterprise-grade **Information Security Policy Document** aligned with **ISO/IEC 27001:2022 (Annex A.5 & A.8)** and **NIST Cybersecurity Framework (PR.AC)** standards.

The policy governs identity and access management (IAM), password enforcement, endpoint encryption, acceptable use of IT assets, and security incident escalation protocols.

---

## 🛠️ Key Policy Domains & Controls Covered
- **Access Control & Identity Management (ISO 27001 A.5.15 / A.8.5):** Role-Based Access Control (RBAC), Least Privilege principle, Privileged Access Management (PAM), and 12-character complex password specifications with mandatory Multi-Factor Authentication (MFA).
- **Acceptable Use Policy - AUP (ISO 27001 A.8.1 / A.8.7):** Mandatory BitLocker full-disk endpoint encryption, Group Policy (GPO) enforcement against unauthorized USB/removable media, and email security guidelines.
- **Incident Response & Escalation (ISO 27001 A.5.24):** Defined escalation paths requiring security breaches or compromised endpoints to be reported within 1 hour to the SOC/IT Security team.
